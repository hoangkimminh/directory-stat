## Installation
### With `npm`
```
$ npm install -g directory-stat
```
### With `yarn`
```
$ yarn global add directory-stat
```

## Usage
```
$ directory-stat --help
Directory statistics

USAGE
  $ directory-stat DIR

ARGUMENTS
  DIR  directory

OPTIONS
  -h, --help     show CLI help
  -v, --version  show CLI version

DESCRIPTION
  Write directory statistics to .dirstat file
```
